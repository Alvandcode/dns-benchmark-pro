
# 🚀 DNS Benchmark Pro

## 📌 معرفی پروژه

DNS Benchmark Pro یک ابزار برای تست و مقایسه سرعت DNS سرورها است که به شما کمک می‌کند بهترین DNS را از نظر سرعت، پایداری و میزان خطا انتخاب کنید.

---

## ✨ قابلیت‌ها

- تست سریع DNSها به صورت همزمان
- پشتیبانی از DNS (UDP)
- ساختار آماده برای DoH و DoT
- محاسبه دقیق عملکرد:
  - میانگین زمان پاسخ
  - میانه (Median)
  - صدک 95 (P95)
  - Packet Loss
- سیستم امتیازدهی برای رتبه‌بندی DNSها
- خروجی گرفتن از نتایج:
  - CSV
  - JSON
  - HTML

---
